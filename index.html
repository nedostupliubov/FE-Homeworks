Часть 1

Селекторы - это виды шаблонов, используемые для выбора элемента, который необходимо стилизовать.

    Виды селекторов
  1. Универсальный - соответствует любому HTML-элементу
  2. Селектор элемента - позволяет стилизовать все элементы данного типа
  3. Селектор идентификатора - позоволяет форматировать один конкретный элемент
  4. Селектор потомка - применяет стили к элементам, расположенным внутри элемента контейнера
  5. Дочерний селектор - применяет стили только если дочерний элемент идет сразу за родительским и между ними нет других элементов
  6. Сестринский - стилизует все элементы из группы одного уровня
  7. Селектор атрибута - стилизует элементы, выбранные на основе имени атрибута
  8. Селектор класса - задает стиль для одного и более элементов с одинаковым именем класса
